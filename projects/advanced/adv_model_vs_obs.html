Advanced: How’s it going? Model output vs observations¶

Description¶

This project begins where the intermediate level “How’s it going? Model Output vs Observations” left off. Students will extend their work by comparing the output from several models against an area averaged observations from a comparable area. Statistics using xarray (built-in, xskillscore, or scipy with apply_ufunc)? Introduce notebook widgets.
